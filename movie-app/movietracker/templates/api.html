API DOCUMENTATION

MovieTracker website provides API to access the data.

GET movie list

endpoint: https://trackmovies.herokuapp.com/api/v1/movies

GET movie

endpoint: https://trackmovies.herokuapp.com/api/v1/movies/[movie_id]

movie_id parameter must be provided to get data of that particular movie.

Authentication

In order to use the API services, users must provide their username and password for authentication. Only the users having account can use API.
